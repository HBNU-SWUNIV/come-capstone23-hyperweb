## <u>프로젝트 소개</u>
  ### 제목
  반려동물에게 맞춤형 자연식 레시피를 추천해주는 SNS 플랫폼

- ### 개요
![그림1](https://github.com/user-attachments/assets/1c56346d-62eb-4585-865b-98f1d1386583)

반려동물의 수가 증가함에 따라 시장도 확대되고 있습니다. 그러나 그에 따라 사료의 영양 균형이나 위생면에서의 문제도 지적되고 있습니다.
그러다 보니 자연식에 관심을 갖는 주인들이 늘고 있고, 이번 프로젝트는 사용자들에게 레시피를 추천하는 SNS 서비스를 제공하기 위해 제작하였습니다.

 ## 🌟 주요 기능 (Key Features)

- 🐾 **반려동물 프로필 등록 및 회원 기능**
  - 반려동물의 이름, 나이, 체중, 알레르기 등 건강 정보를 등록
  - 사용자 계정 생성 및 로그인 기능

- 🍽️ **일일/한 달 식단 자동 추천**
  - 반려동물의 특성에 따라 맞춤형 레시피를 자동 추천
  - 1일 단위, 30일 단위 식단 구성 가능

- 🧠 **생성형 AI 기반 가이드 체험**
  - 실제 데이터를 기반으로 한 레시피 추천 흐름을 직접 체험 가능
  - 더미 데이터를 활용해 미리보기 UX 제공

- 📝 **SNS 스타일의 레시피 공유 플랫폼**
  - 다른 사용자들과 레시피를 공유

- 📊 **영양소 분석 및 시각화**
  - 레시피의 영양 균형을 시각적으로 확인 가능
  - 부족한 영양소를 쉽게 파악하여 개선 유도

- 💬 **직관적인 UI/UX**
  - 접고 펼 수 있는 반응형 사이드바 탑재
  - 누구나 쉽게 접근 가능한 깔끔한 인터페이스
